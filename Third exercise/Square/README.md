# question
Write a program that takes the number n from the input and then prints a hollow square of length n and width n with stars.
# input
You are given the number n on a line.
# output
Print the resulting square in the output.
# example:

## Sample input 1:
```
6
```
## Sample output 1:
```
******
*    *
*    *
*    *
*    *
******
```
